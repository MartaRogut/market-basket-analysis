# Market Basket Analysis

This project focuses on market basket analysis using transaction data.
The goal is to identify relationships between products that are frequently bought together.

## Technologies
- Python
- pandas
- mlxtend

## Description
The analysis is based on transactional data transformed into a binary format.
Association rules are generated to discover frequent itemsets and product combinations.

## Files
- `market-basket-analysis.py` – main analysis script
- `basket_analysis1.csv` – sample dataset
